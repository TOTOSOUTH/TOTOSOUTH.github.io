HTML5

一、定义

是用来描述网页的一种语言，被称为超文本标记语言。标记语言是一套标记标签。标签是由尖括号包围的关键字

  • 单标签：

    e.g. <img>

  • 双标签：

    e.g. <html></html>

二、 DOCTYPE声明

<!doctype html> 位于文档最前面，处于标签之前，避免浏览器的怪异模式

三、基本骨架

SEO是搜索引擎英文缩写，通过对网站的调整，满足搜索引擎的排名需求(广告除外)

四、标题标签

1. 标题介绍与应用

标题是通过 <h1> - <h6> (双标签) 标签进行定义的

<h1>定义最大标题
<h6>定义最大标题

快速生成六个标题标签快捷键：==h$*6==

正确使用标签

1. 请确保html标题标签只用于标题  
2. 不要仅仅为了生成加粗体或大号文本使用标题
3. 正确使用标题有益于SEO
4. 根据重要程度定义标题级别

2. 标题标签位置摆放

在标签中添加属性：

align = “left || center || right” 默认居左

五、标签之段落、换行、水平线

1. 段落标签

段落是通过<p>标签定义的

2. 换行标签

在不产生一个新段落的前提下换行，使用<br>

<p>在一个段落中<br>换行</p>

3. 水平线

<hr/>标签在HTML页面中创建水平线

<hr color = "" width = "" size = "" allign = "" />

  • 属性：

    1. color：设置水平线颜色
    2. width：设置水平线宽度
    3. size：设置水平线高度
    4. align：设置水平线对齐方式（默认居中），可取值 left || right

六、图片

1、图片的标签

< img > 标签定义HTML页面中图像

<img str = "" alt = "" title = "" width = "" height = "">

  • 注： 是单标签，无需进行闭合操作
  • 属性
    1. src：路径（图片地址与名字）
    2. alt：规定图像的代替文本
    3. width：规定图像的宽度
    4. height：规定图像的高度
    5. title：鼠标悬停在图像上给予提示

2、图片路径

  1. 绝对路径
  2. 相对路径
    1. 子级关系：/
    2. 父级关系：../
    3. 同级关系: ./ (可省略)
  3. 网络路径（直接复制网址）

七、超文本链接

1. 超链接标签

HTML使用标签<a>来设置超文本链接（双标签）

<a href = "ur1">链接文本</a>

  • 属性
    在标签<a>中使用了 href 属性来描述链接地址，默认情况下，链接将以以下形式出现在浏览器中：
    1. 一个未访问过的链接显示为蓝色字体并带有下划线
    2. 访问过的链接显示为紫色字体并带有下划线
    3. 点击链接时，链接显示为红色字体并带有下划线
  • 可通过css样式修改以上效果

2. 超链接表现

鼠标悬停在上方时，鼠标样式改变

八、文本标签

1. 常用文本标签

标签 描述
<em> 定义着重文字
<b> 定义加粗文字
<i> 定义斜体字
<strong> 定义加重语气
<del> 定义删除字
<span> 元素没有特定含义
  • 常用文本标签和段落是不同的，段落代表一段文本，而文本标签一般表示文本词汇

九、 列表标签

1. 有序列表

有序列表是一列项目，列表项目使用数字进行标记。有序列表始于<ol>标签。每个列表项始于<li>标签。

  • type属性

    <ol>的属性type拥有的选项

    1. 1 表示列表项目用数字标号（1，2，3 …）
    2. a 表示列表项目用小写字母标号（a，b，c …）
    3. A 表示列表项目用大写字母标号（A，B，C …）
    4. i 表示列表项目用小写罗马数字标号（i，ii，iii …）
    5. I 表示列表项目用大写罗马数字标号（I，II，III …）

  • 有序列表嵌套

2. 无序列表

无序列表是一个项目列表，此项目使用粗体圆点进行标记
无序列表始于<ul>标签。每个列表始于<li>标签

  • type属性

    <ul>的属性type拥有的选项

    1. disc：默认实心圆
    2. circle：空心圆
    3. square：小方块
    4. none：不显示

  • 无序列表嵌套

  • 导航效果（通过css 改变样式）

  • 快捷键

    快速生成 ul + li 的布局：==ul>li*n==（n为数字，根据需要li数量更改）

十、表格

1. 表格标签

  • 表格：<table>
  • 行：<tr>
  • 列：<td>

  • 快捷键：

    快速生成表格结构：==table>trn>tdm{文本信息}==

  • 表格属性

    1. border：设置表格的边框
    2. width：设置表格的宽度
    3. height：设置表格的高度

2. 表格单元格合并

  • 水平合并：colspan （保留左边删除右边）
  • 垂直合并：rowspan （保留上边删除下边）

十一、Form表单

1. 表单组成

表单在Web网页中用来给用户填写信息，从而能采集用户信息，使网页具有交互功能。
所有用户输入内容的地方都用表单来写，如登陆注册、搜索框

  • 表单是由容器和控件组成的，一个表单一般应该包括用户填写信息的输入框、按钮等，这些输入框、按钮叫做==控件==，表单就是==容器==，它能容纳各种各样的控件。

    <form action = "url" method = "get|post" name = "myform"></form>

  • 属性说明：

    1. action：服务器地址
    2. name：表单名称
    3. method 中 get 和 post 的区别
      1. 数据提交方式，get 把提交的数据 url可以看到，post 看不到
      2. get 一般用于提交少量数据，post 用来提交大量数据

  • 表单元素
    一个完整的表单包含三个基本组成部分：表单标题、表单域、表单按钮

    1. 表单标签
    2. 表单域
    3. 表单按钮

2. 表单元素

  • 文本框
    文本域通过<input type = "text">标签来设定，当用户要在表单中输入字母、数字等内容时，就会用到文本域

  • 密码框
    密码字段通过标签<input type = "password">来定义

    • 密码字段字符不会明文显示，而是用星号或者圆点代替

  • 提交按钮
    当用户单击确认按钮时，表单的内容会被传送到另一个文件。表单的动作属性定义了目的文件的文件名。由动作属性定义的这个文件通常会对接收到的输入数据进行相关的处理

    <input type = "submit" value = "你好"> value 替换按钮文本

十二、块元素与行内元素（内联元素）

  • 区别
块级元素 内联元素
块元素会在页面中独占一行（自上向下垂直排列） 行内元素不会独占页面中的一行，只占自身的大小
可以设置 width，height 属性 行内元素设置 width，height 属性无效
一般块级元素可以包含行内元素和其他块级元素 一般行内元素包含行内元素不包含块级元素

  • 常见块级元素

    div、form、h1~h6、hr、p、table、ul 等

  • 常见内联元素

    a、b、em、i、span、strong 等

  • 行内块状元素（特点：不换行、能够识别宽高）

    button、img、input 等
