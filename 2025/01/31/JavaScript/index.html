ECMAScript

一、JavaScript组成

  • JavaScript
    • ECMAScript（JavaScript语言基础）
    • Web APls
      • DOM （页面文档对象模型）
      • BOM （浏览器对象模型）
  • ECMAScript：
    规定了 js 基础语法核心知识
    • 比如：变量、分支语句、循环语句、对象等等
  • Web APls：
    • DOM 操作文档，比如对页面元素进行移动、大小、添加删除等操作
    • BOM 操作浏览器，比如页面弹窗，检测窗口宽度、存储数据到浏览器等等

参考网站：https://developer.mozilla.org/zh-CN/docs/Web/JavaScript

二、JavaScript书写顺序

1. 内部JavaScript

  • 直接写在 HTML 文件里，用 script 标签包住
    • 注：script 标签写在 </body> 上面

2. 外部JavaScript

  • 代码写在以 .js 结尾的文件夹里

    <script src="xx.js"></script>

    • 此时标签中间无需写代码，会被忽略，写了也不执行

3. 内联JavaScript

  • 代码写在标签内部

三、JavaScript输入输出语法

1. 输出语法

  • 输出语法

    1. document.write('Hello')

    • 向 body 内输出内容
    • 如果输出内容写的是标签，也会被解析成网页元素

    1. alert('Hello')

    • 页面弹出警示对话框

    1. console.log('控制台输出')

    • 控制台输出语法，程序员调试使用

2. 输入语法

  • 输入语法

    prompt('请输入')

    • 显示一个对话框，对话框中包含一条文字信息，用来提示用户输入文字

3. 输入输出执行顺序

  • alert('')和prompt('')会跳过页面渲染先被执行

四、变量

1. 定义

  • 变量是计算机存储数据的“容器”

2. 基本使用

  • 声明变量

    let 变量名

    • 定义多个变量用’,’隔开
    • 不可重复声明

  • 变量赋值

    变量名 = 字面量

    • 声明时可直接赋值 叫做初始化

  • 更新变量

    变量名 = 新字面量

  • var 与 let 的区别

    • 可以先使用 再声明
    • var 声明的变量可以重复声明
    • 比如变量提升全局变量、没有块级作用域等问题

3. 本质

  • 是程序在内存中申请的一块用来存放数据的小空间

4. 命名规则和规范

  • 规则：
    • 不能用关键字
      • 例如：let、var …
    • 只能用下划线、数字、字母、$ 组成
      • 数字不能开头
    • 严格区别大小写
  • 规范：
    • 望文生义
    • 小驼峰命名法

5. 数组

  • let 变量名 = []

  • let 变量名 = [数据1, 数据2, 数据3, ... , 数据n]

  • 数组新增
    • arr.push (返回值是新数组长度) 插在结尾

      arr.push(数据1, 数据2, 数据3, ... , 数据n)

    • arr.unshift 插在开头
  • 数组删除
    • arr.pop() 删除末位一个 (返回值是删除的元素)
    • arr.shift() 删除第一个 (返回的是删除的元素)
    • arr.splice(操作的下标, 删除的个数) 如果省略个数 默认删到最后

五、常量

  • 使用 const 声明的变量称为”常量”
  • 不允许重新赋值
  • 用const声明的对象和数组可以追加里面的值

六、数据类型

  • 基本数据类型
  • 引用数据类型

1. 基本数据类型

  • number 数字型
    • 数学中所有数字
    • NaN代表一个计算错误 它是一个不正确的或者一个未定义的数学操作所得到的结果
  • string 字符串型
    • 通过单引号(‘)、双引号(“)或反引号(`)包裹的数据都叫字符串，单引号和双引号没有本质上的区别，推荐使用单引号
    • 必要时可以使用转义符”", 输出单引号或者双引号
    • 模板字符串
      • 外面用反引号(``)包含
      • 里面用(${})包含所使用变量
  • boolean 布尔型
  • undefined 未定义型
    • 只定义 未赋值
  • null 空类型

2. 引用数据类型

  • object 对象

七、检测数据类型

  • 通过 typeof 检测数据类型
    • 作为运算符：typeof x
    • 函数形式：typeof(x)

八、类型转换

1. 隐式转换

  • 某些运算符被执行时，系统内部自动将数据类型转换，这种转换被称为隐式转换
  • 规则：
    • +号两边只要有一个是字符串，都会把另一个转成字符串
    • 除了 + 以外的算术运算符 都会把数据转成数字类型
  • 缺点：转换类型不明确
  • +号作为正号解析可以转换成数字型
  • 任何数据和字符串相加结果都是字符串

2. 显示转换

  • 转换为数字型
    • Number()
    • parseInt() 只保留整数 不四舍五入 只识别字符串开头数字
    • parseFloat() 保留小数 只识别字符串开头数字

九、运算符

1. 一元运算符

  • 运算中只有一个数字
  • ‘+’ ‘-‘ 正负号
  • 自增运算符

2. 比较运算符

  • ‘===’：左右两边是否类型和值都相等
  • ‘!==’：左右两边是否不全等
  • 比较运算符有隐式转换

十、函数

  • 函数返回多个值使用数组

    return [max, min]

  • 实参过多：不被使用 仅使用前与形参数量相同实参

  • 形参过多：为赋值形参为 undefined

1. 匿名函数

  • 没有函数名

    let fn = function () {}

2. 立即执行函数

  • 无需调用 直接执行
  • 防止变量污染
  • 立即执行函数必须加分号’;’

(function() {})();

  • 最后一个 ‘()’ 相当于调用函数 内放实参

(function() {} ());

3. 逻辑中断

  • 为防止无实参传递给函数 可给予函数形参默认值

function fn(x = 0, y = 0) {}

十一、对象

  • 对象（object）：JavaScript里的一种数据类型
  • 一种无序的数据集合（数组是有序的数据集合）

1. 对象使用

let 对象名 = {}
let 对象名 = new Object()

  • 对象有属性和方法组成
  • 查

    对象名.属性
    对象名[‘属性名’]

  • 改

    对象名.属性 = 新值

  • 增

    对象名.新属性 = 新值

  • 删

    delete 对象名.属性

2. 遍历对象

3. 内置对象

  • Math
    • 提供了一系列数学运算方法
    • 包含：
      • random：生成 [0, 1) 之间的随机数
        • 生成 0-10 的随机数

          Math.floor(Math.random() * (10 + 1))

        • 取到N~M的随机整数

总结：如果文本内容中包含 html 标签时推荐使用 innerHTML，否则建议使用 innerText 属性。

四、操作元素属性

有3种方式可以实现对属性的修改：

1. 常用属性修改

  • 直接能过属性名修改，最简洁的语法

2. 控制样式属性

  • 应用【修改样式】，通过修改行内样式 style 属性，实现对样式的动态修改。

通过元素节点获得的 style 属性本身的数据类型也是对象，如 box.style.color、box.style.width 分别用来获取元素节点 CSS 样式的 color 和 width 的值。

任何标签都有 style 属性，通过 style 属性可以动态更改网页标签的样式，如要遇到 css 属性中包含字符 - 时，要将 - 去掉并将其后面的字母改成大写，如 background-color 要写成 box.style.backgroundColor

  • 操作类名(className) 操作CSS

如果修改的样式比较多，直接通过style属性修改比较繁琐，我们可以通过借助于css类名的形式。

注意：

1.由于class是关键字, 所以使用className去代替

2.className是使用新值换旧值, 如果需要添加一个类,需要保留之前的类名

  • 通过 classList 操作类控制CSS

为了解决className 容易覆盖以前的类名，我们可以通过classList方式追加和删除类名

3. 操作表单元素属性

表单很多情况，也需要修改属性，比如点击眼睛，可以看到密码，本质是把表单类型转换为文本框

正常的有属性有取值的跟其他的标签属性没有任何区别

获取:DOM对象.属性名

设置:DOM对象.属性名= 新值

4. 自定义属性

标准属性: 标签天生自带的属性 比如class id title等, 可以直接使用点语法操作比如： disabled、checked、selected

自定义属性：

在html5中推出来了专门的data-自定义属性

在标签上一律以data-开头

在DOM对象上一律以dataset对象方式获取

五、间歇函数

知道间歇函数的作用，利用间歇函数创建定时任务。

setInterval 是 JavaScript 中内置的函数，它的作用是间隔固定的时间自动重复执行另一个函数，也叫定时器函数。
