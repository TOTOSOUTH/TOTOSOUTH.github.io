一、JavaScript组成

  • JavaScript
    • ECMAScript（JavaScript语言基础）
    • Web APls
      • DOM （页面文档对象模型）
      • BOM （浏览器对象模型）
  • ECMAScript：
    规定了 js 基础语法核心知识
    • 比如：变量、分支语句、循环语句、对象等等
  • Web APls：
    • DOM 操作文档，比如对页面元素进行移动、大小、添加删除等操作
    • BOM 操作浏览器，比如页面弹窗，检测窗口宽度、存储数据到浏览器等等

参考网站：https://developer.mozilla.org/zh-CN/docs/Web/JavaScript

二、JavaScript书写顺序

1. 内部JavaScript

  • 直接写在 HTML 文件里，用 script 标签包住
    • 注：script 标签写在 </body> 上面

2. 外部JavaScript

  • 代码写在以 .js 结尾的文件夹里

    <script src="xx.js"></script>

    • 此时标签中间无需写代码，会被忽略，写了也不执行

3. 内联JavaScript

  • 代码写在标签内部

三、JavaScript输入输出语法

1. 输出语法

  • 输出语法

    1. document.write('Hello')

    • 向 body 内输出内容
    • 如果输出内容写的是标签，也会被解析成网页元素

    1. alert('Hello')

    • 页面弹出警示对话框

    1. console.log('控制台输出')

    • 控制台输出语法，程序员调试使用

2. 输入语法

  • 输入语法

    prompt('请输入')

    • 显示一个对话框，对话框中包含一条文字信息，用来提示用户输入文字

3. 输入输出执行顺序

  • alert('')和prompt('')会跳过页面渲染先被执行

四、变量

1. 定义

  • 变量是计算机存储数据的“容器”

2. 基本使用

  • 声明变量

    let 变量名

    • 定义多个变量用’,’隔开
    • 不可重复声明

  • 变量赋值

    变量名 = 字面量

    • 声明时可直接赋值 叫做初始化

  • 更新变量

    变量名 = 新字面量

  • var 与 let 的区别

    • 可以先使用 再声明
    • var 声明的变量可以重复声明
    • 比如变量提升全局变量、没有块级作用域等问题

3. 本质

  • 是程序在内存中申请的一块用来存放数据的小空间

4. 命名规则和规范

  • 规则：
    • 不能用关键字
      • 例如：let、var …
    • 只能用下划线、数字、字母、$ 组成
      • 数字不能开头
    • 严格区别大小写
  • 规范：
    • 望文生义
    • 小驼峰命名法

5. 数组

  • let 变量名 = []

  • let 变量名 = [数据1, 数据2, 数据3, ... , 数据n]
